# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-29

### Features
- Save WhatsApp and WhatsApp Business statuses to device gallery
- View images and videos in separate tabs
- Light/Dark mode toggle
- Multi-language support (Localization)
- Minimalistic and intuitive design
- Android 10+ support with ManageStorage permission
