
# Whatsapp Status Saver     

A Flutter application designed to save WhatsApp and WhatsApp Business statuses directly to your device's gallery.

## Features

- Personal/Business Whatsapp Statuses
- Light/dark mode toggle
- Localization
- Minimalistic Design
